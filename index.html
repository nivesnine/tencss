Navigation

You can make the navigation "fixed" by including the .fixed class.

Use .container to keep things contained.

Use .dropdown is used for the first level of submenus, and .drilldown for the second.

This menu works on mobile as well!

<div class="menu-icon">☰</div>
<nav class="navigation fixed">
  <ul class="container">
    <li><a href="#forms">Forms</a></li>
    <li><a href="#cards">Cards</a></li>
    <li>
      <a href="#">More Stuff</a>
      <ul class="dropdown">
        <li><a href="#navigation">Navigation</a></li>
        <li><a href="#notifications">Notifications</a></li>
        <li>
          <a href="#">Even More Stuff</a>
          <ul class="drilldown">
            <li><a href="#breadcrumbs">Breadcrumbs</a></li>
            <li><a href="#sizes">Sizes</a></li>
          </ul>
        </li>
      </ul>
    </li>
  </ul>
</nav>
 
 
 
 
 
 
 
 
 
 
 

Grid

The grid system has 4 screen size targets col-*, sm-col-*, md-col-*, and lg-col-* all of which span 0 to 10 columns.

For example, if you want to show content only for mobile, you could use .col-10 sm-col-0. Check out how the first column in the example code switches.

Forms

Forms can have columns! Just remember to include the .row class in your form tag!

The .form class adds a border and box-shadow, you don't like it, don't use it!

The .file class and .checkbox class can help you display these input types in a slightly nicer way.

<form class="form row">
  <div class="col-10">
    <fieldset>
      <label>Text</label>
      <input type="text" name="text">
      <label>Number</label>
      <input type="number" name="text">
    </fieldset>
  </div>
  <div class="col-5">
    <label>Select</label>
    <select>
      <option>Option 1</option>
      <option>Option 2</option>
      <option>Option 3</option>
      <option>Option 4</option>
    </select>
  </div>
  <div class="col-5">
    <label>File</label>
    <label class='file' for="file">Upload File</label>
    <input id='file' type="file">
  </div>
  <div class="col-10">
    <label class="checkbox">Checkbox</label>
    <input type="checkbox"> 
  </div>
  <div class="col-4">
    <label>Datetime-Local</label>
    <input type="datetime-local">
  </div>
  <div class="col-6">
    <label>Tel</label>
    <input type="tel">
  </div>
  <div class="col-7">
    <label>Email</label>
    <input type="email">
  </div>
  <div class="col-3">
    <label>Password</label>
    <input type="password">
  </div>
</form>

Cards

The .cards class can be used on pretty much any element to give it a box-shadow.

Cards have a few subclasses. There's a .card-image if you need an image. .card-header goes at the top, .card-content goes in the middle, and .card-footer goes at the buttom. If you need buttons instead of a footer use .card-footer-buttons and specify .one-button, .two-buttons, or .three-buttons.

<div class="col-10 md-col-5">
  <div class="card">
    <div class="card-image">
      <img src="https://via.placeholder.com/600">
    </div>
    <div class="card-content">
      <p>This is just some text about an image</p>
    </div>
    <div class="card-footer">
      <p>This is just some footer text</p>
    </div>
  </div>
</div>

<div class="col-10 md-col-5">
  <div class="card">
    <div class="card-header">
      <h4>This is a card header</h4>
    </div>
    <div class="card-image">
      <img src="https://via.placeholder.com/600">
    </div>
    <div class="card-content">
      <p>Here's some content for this card!</p>
    </div>
    <div class="card-footer-buttons two-buttons">
      <button>Go Here</button>
      <button>Go There</button>
    </div>
  </div>
</div>
